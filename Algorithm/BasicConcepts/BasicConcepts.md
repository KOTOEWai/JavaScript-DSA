#  What is an Algorithm?

> An **Algorithm** is a step-by-step set of instructions designed to solve a specific problem or perform a particular task. It is the "logic" that tells the computer exactly what to do with the data it receives.

---

##  1. Real-World Analogy: A Cooking Recipe
The easiest way to understand an algorithm is by looking at a recipe for baking a cake.

###  Input (Ingredients)
* Flour
* Eggs
* Sugar
* Butter

###  The Algorithm (Process)
1. **Preheat** the oven.
2. **Mix** the dry ingredients.
3. **Add** the wet ingredients.
4. **Bake** for 30 minutes.

###  Output
* A delicious finished cake! 🍰

---

##  2. Core Components
Every algorithm follows this basic structure:

| Component | Description |
| :--- | :--- |
| **Input** | The data required to start the process. |
| **Steps** | The specific, ordered actions to take. |
| **Output** | The final result or solution. |

---
