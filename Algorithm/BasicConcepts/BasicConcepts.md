# What is an Algorithm?


* An Algorithm is a step-by-step set of instructions designed to solve a specific problem or perform a particular task. In computer science, it is the "logic" that tells the computer exactly what to do with the data it receives.



* 1. Real-World Analogy: A Cooking Recipe
The easiest way to understand an algorithm is by looking at a recipe for baking a cake.

    * Ingredients: Flour, Eggs, Sugar, Butter.


    * The Algorithm (Steps): 
        *1. Preheat the oven.
        *2. Mix the dry ingredients.
        *3. Add the wet ingredients.
        *4. Bake for 30 minutes.

    * Output: A finished cake.
